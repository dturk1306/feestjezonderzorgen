Looker Studio rapport verbinden

Maak een gratis Looker Studio rapport op basis van je GA4 data en plak de embed-URL hieronder. Zie de setup-instructies voor stap-voor-stap uitleg.

⚠ Rapport kon niet geladen worden Controleer de volgende punten:
1. Rapport is openbaar: Open Looker Studio → Bestand → Rapport delen → stel in op "Iedereen met de link kan bekijken".
2. Gebruik de embed-URL: Klik op Delen → Rapport insluiten → kopieer de URL die begint met lookerstudio.google.com/embed/reporting/…
3. Verwijder de verbinding hieronder en plak de juiste URL opnieuw.

settings Setup-instructies (±30 min)

Volg deze stappen om van lokale events naar echte cross-device, real-time data te gaan.

✓

Website live + UTM-tags geconfigureerd

Website staat live op dturk1306.github.io/feestjezonderzorgen/. UTM-tags actief voor WhatsApp, email, Instagram bio en LinkedIn.

✓

Instagram pagina aangemaakt

Actief op @feestjezonderzorgen. Zet het account om naar een Zakelijk of Creator-profiel om toegang te krijgen tot Inzichten en link-tracking.

✓

Tracking-code in index.html

De fzzTrack()-functie staat in index.html en slaat events op in localStorage én stuurt ze door naar GA4 (zodra GA4 gekoppeld is). Events: quiz_start, quiz_answer, quiz_complete, share_click, steun_click, checklist_print, checklist_copy.

4

GA4 property aanmaken

Ga naar analytics.google.com → Beheer (tandwiel) → + Property aanmaken → selecteer Web → vul de website-URL in. Noteer je Measurement ID (formaat: G-XXXXXXXXXX).

5

GA4 Measurement ID toevoegen aan index.html

Open index.html en vervang de twee plekken waar G-XXXXXXXXXX staat door jouw echte Measurement ID. Zoek op G-XXXXXXXXXX in de broncode — je vindt het in de GA4-scripttags bovenaan.

<!-- GA4 — vervang G-XXXXXXXXXX door jouw Measurement ID -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
6

Custom events verifiëren in GA4

Na 24-48 uur verschijnen de events in GA4 → Rapporten → Betrokkenheid → Events. Je ziet:

  • quiz_start — quiz-sectie scrolt in beeld
  • quiz_answer — antwoord gegeven (met question, correct)
  • quiz_complete — quiz afgerond (met score en total)
  • share_click — gedeeld, met parameter channel (whatsapp / checklist_whatsapp / email / native / copy)
  • steun_click — steun-knop geklikt
  • checklist_print / checklist_copy — checklist gedownload
7

Looker Studio rapport aanmaken

Ga naar lookerstudio.google.com → Leeg rapport → Gegevensbron toevoegen → Google Analytics → selecteer je FZZ property. Voeg metrics toe: Sessies, Actieve gebruikers, Events (per event-naam), UTM-campagne-uitsplitsing. Publiceer het rapport → Delen → Insluiten → kopieer de embed-URL → plak in het veld hierboven.
